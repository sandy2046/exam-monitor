## 使用方法

### 1. 同步模板
在应用首页点击"同步远程模板"按钮，系统会自动从 `/public/templates/cet-templates.json` 加载模板。

### 2. 下载到本地
- 在模板列表中找到 "CET-4 考试流程" 或 "CET-6 考试流程"
- 点击"下载"按钮
- 模板将保存到本地存储，可离线使用

### 3. 启动考试
1. 在"已下载模板"中选择对应的 CET 模板
2. 点击"使用"按钮
3. 设置实际开始时间（如：CET-4 设置为当天 8:30）
4. 点击"确认开始考试"

### 4. 考试监控
- 系统会自动根据时间推进到对应阶段
- 每个阶段会显示：
  - 阶段名称和时间范围
  - 详细描述
  - 注意事项（橙色区域）
  - 进度百分比和已过时间
  - 下一阶段预告

每个阶段都有对应的简短注意事项，显示在橙色警告框中：

- **⚠️ 注意事项**: 核对证件、安排座位、提示试音
- **⚠️ 注意事项**: 检查印刷、指导粘贴条形码
- **⚠️ 注意事项**: 强调不得翻阅试题册
- **⚠️ 注意事项**: 不得打开试题册，不戴耳机
- **⚠️ 注意事项**: 命令打开试题册，戴上耳机
- **⚠️ 注意事项**: 清点答题卡1，确认无误
- **⚠️ 注意事项**: 核对信息、记录缺考情况
- **⚠️ 注意事项**: 维持考场秩序，关注异常
- **⚠️ 注意事项**: 提醒时间，停止答题准备
- **⚠️ 注意事项**: 清点试卷、整理资料

## 自定义模板

如需修改模板，可以：
1. 编辑 `/public/templates/cet-templates.json` 文件
2. 或者在应用中下载模板后创建本地副本进行编辑

## 数据来源

模板数据基于官方考试流程：
- CET-4: 全国大学英语四级考试流程
- CET-6: 全国大学英语六级考试流程

如有特殊要求，可根据实际情况调整时间偏移（offset）和注意事项。
